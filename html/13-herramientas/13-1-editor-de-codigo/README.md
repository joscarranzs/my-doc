# Editor de código

Herramientas para escribir HTML eficientemente.

## Editores recomendados

### Visual Studio Code
- **Extensiones útiles**:
  - HTML CSS Support
  - Auto Rename Tag
  - IntelliSense for CSS class names
  - Live Server
  - Prettier

### Sublime Text
- Ligero y rápido
- Paquetes para HTML
- Multi-cursor editing

### Atom `(descontinuado)`
- GitHub's editor
- Buena integración con Git

## IDEs completos

### WebStorm
- IDE profesional
- Debugging integrado
- Refactoring avanzado

### Brackets
- Enfocado en web
- Live preview
- Preprocesadores integrados

## Editores en línea

### CodePen
- Prototipado rápido
- Compartir código
- Comunidad activa

### JSFiddle
- Similar a CodePen
- Enfoque en JavaScript

### Replit
- Entorno completo
- Colaboración en tiempo real

**Consejo**: VS Code es excelente para principiantes. Aprende atajos de teclado para aumentar productividad.
