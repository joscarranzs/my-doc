# Headings y jerarquía

## Explicación

Headings estructuran el contenido para SEO y accesibilidad.

## Ejemplo básico

```html
<h1>Título principal</h1>
<h2>Subtítulo</h2>
<h3>Subsubtítulo</h3>
```

## Ejemplo de la vida real

En un artículo.

```html
<article>
    <h1>Los beneficios del ejercicio</h1>
    <h2>Beneficios físicos</h2>
    <p>...</p>
    <h2>Beneficios mentales</h2>
    <p>...</p>
    <h3>Reducción del estrés</h3>
    <p>...</p>
</article>
```

## Notas importantes

- Solo un h1 por página.
- Jerarquía lógica.

## Mejores prácticas

- Incluye palabras clave en headings.
- No uses para estilo.
